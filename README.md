## 🚀 Mealify Restaurant Website (Kandil)

This is a modern Restaurant Landing Page built as a front-end practice project.
The project focuses on building a clean food website layout, implementing responsive design, and creating engaging UI animations to enhance user experience.
The goal of this project was to practice advanced CSS styling, responsive layouts,
and interactive UI elements commonly used in restaurant and food-related websites.

------

## 🛠 Built With

✔️ HTML

✔️ CSS

✔️ Font Awesome

✔️ Google Fonts


------

## 🎯 Key Features & Concepts Practiced

✔️ Semantic HTML structure for clean and accessible markup

✔️ Fully responsive layout for mobile, tablet, and desktop

✔️ Modern Hero Section with call-to-action buttons

✔️ Restaurant menu section with styled food cards

✔️ Chefs section with interactive hover effects

✔️ Gallery section with responsive image grid

✔️ Contact section with embedded Google Map

✔️ Smooth UI animations and hover effects

✔️ CSS Flexbox for layout alignment

✔️ CSS Grid for structured gallery layout

✔️ Custom typography using Google Fonts

✔️ Icon integration using Font Awesome

✔️ Clean spacing, alignment, and visual hierarchy

------

## 🧠 Architecture Approach

✔️ Section-based layout architecture

✔️ Flexbox and Grid combined for flexible layout control

✔️ Reusable CSS classes for scalable styling

✔️ Responsive design using media queries

✔️ Clear separation between layout sections
